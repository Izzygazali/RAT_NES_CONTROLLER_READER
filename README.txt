Author: Tyler Starr and Ezzeddeen Gazali

-The RATPUTER folder contains the HDL project for the NES_CONTROLLER_INTERFACE project.
-The Assembly folder contains the commented assembly code associated with the project.
-The Python Script contains the PC intrepter for the project. It requies python and 3 dependecies
to run: PyKeyboard, serial, and binascii.
-Information about physical hookup of our NES controller can be found on our instructables page.
-The NES_CONTROLLER_INTERFACE.bit can be uploaded to a Basys 3 board to demo the project if desired.
